# Research_Agent
大语言模型（LLM）正从“能对话”向“能做事”演进。在科研辅助、深度学习、商业调研等高阶知识工作场景中，用户提出的问题往往具有多跳推理、信息验证、工具依赖等特征——仅靠单轮问答无法满足真实需求。  为此，业界提出了 Research Agent 的概念：一个能够自主规划子任务、调用外部工具（如联网搜索、代码执行、知识库检索）、整合多源证据并生成可核查答案的智能体。GAIA（General AI Assistants）基准正是对此类能力的权威评测之一，其题目强调真实世界问题解决能力，要求模型具备推理、浏览、工具调用与答案归一化等综合技能。（arXiv:2311.12983）

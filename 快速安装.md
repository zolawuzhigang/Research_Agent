# 快速安装依赖

## 问题
```
ModuleNotFoundError: No module named 'uvicorn'
```

## 解决方案

### 最简单的方法（复制粘贴到PowerShell）

```powershell
# 安装核心依赖（必需）
python -m pip install uvicorn fastapi pydantic loguru pyyaml requests aiohttp
```

### 如果网络慢，使用国内镜像

```powershell
# 使用清华镜像
python -m pip install -i https://pypi.tuna.tsinghua.edu.cn/simple uvicorn fastapi pydantic loguru pyyaml requests aiohttp
```

### 安装完成后运行

```powershell
python run_server_fast.py
```

## 验证安装

```powershell
python -c "import uvicorn; print('安装成功！')"
```

如果显示 "安装成功！"，说明可以运行了。
